ubuntu server 20.04 设置root用户登录

# 1.设置root密码，已设置的可以往下走

sudo passwd root

# 2.修改文件

sudo vi /etc/ssh/sshd_config

把 PermitRootLogin without-password 改为 PermitRootLogin yes，注意PermitRootLogin without-password可能被注释掉了，要去掉注释 PS：如果2021.06后的20版本没有找到PermitRootLogin without-password，直接文件末尾添加PermitRootLogin yes即可

# 3.重启sshd服务

sudo systemctl restart sshd
